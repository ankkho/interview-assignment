Project setup:

`git clone git@github.com:ankkho/interview-assignment.git `  
`cd interview-assignment`   
`npm i`  
`npm run dev`

Starting docker:  
`cd interview-assignment`  
`docker-compose up`

Run test:  
`npm run test`

Run a migration:  
`npx sequelize db:migrate`

More options:  
`npx sequelize --help`
